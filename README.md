# Q&A Chatbot Web App

A **smart Q&A chatbot** built using Python that simulates human-like conversation. This chatbot responds to predefined queries, making it a simple yet effective way to demonstrate interactive applications. It can easily be extended with AI modules for more advanced conversational abilities.

## Features

* **Text-Based Interaction:** Users can type questions and receive instant responses.
* **Predefined Knowledge Base:** Handles common queries using a structured set of responses.
* **Extendable Design:** Easily add new responses or integrate AI/NLP models for smarter interactions.
* **Lightweight & Fast:** Minimal dependencies, runs smoothly on local machines.

## How It Works

The chatbot operates by matching user inputs against a **predefined set of questions and answers**. When a match is found, it returns the corresponding response. The structure is modular, so additional AI or NLP modules can be integrated to handle more complex or dynamic conversations.

## Tech Stack

* **Backend:** Python with Flask for web interface
* **Frontend:** HTML/CSS for a clean, interactive design
* **Logic:** Python dictionaries or JSON files as the knowledge base

## Installation and Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/mrjawadd/chat_bot.git
   cd chat_bot
   ```
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:

   ```bash
   python app.py
   ```
4. Open your browser and navigate to:

   ```
   http://127.0.0.1:5000
   ```
5. Start typing questions in the chat interface and see the chatbot respond.
